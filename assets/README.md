## Directory Structure

```
assets/
  fullhd/          # 1920x1080 renders and captures
  2k/              # 2560x1440
  4k/              # 3840x2160
  audio/           # Sound files
  video/           # Video recordings
```

## Naming Conventions

```
<type>_<description>_<date>_<resolution>.<ext>

Examples:
  overlay_ghost_wifi_20250304_fullhd.png
  capture_demo_20250304_fullhd.mp4
  audio_ambient_20250304.wav
```
